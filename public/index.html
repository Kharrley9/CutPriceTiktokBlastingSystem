📋 Setup Instructions

  1. Message your bot on Telegram with /start
  2. Create a new Telegram group and add the bot
  3. Make the bot an Admin with all permissions
  4. Send /register in the group
  5. The bot will configure the group automatically
  6. Share the invite link with up to 100 members
  7. Add TikTok links using the Links tab
  8. Links will be blasted daily at the configured time
